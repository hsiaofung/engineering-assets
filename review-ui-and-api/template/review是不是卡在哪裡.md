# Use Case: 
例如 Sprint Planning 被問：
Review 是不是卡在哪裡？

---

# 會議句型:
- 你只要記三段： 狀態 → 原因 → 下一步

## 套用：   

目前還在 review。主要有幾個 API clarification 尚未完成，部分等待 BE 更新 Swagger，完成後 FE 會再驗證。



## 甚至更短：

還在 review，目前有幾個 API dependency 尚未確認，等 BE contract update 後 FE 會驗證。

--- 

## 例如：

1. 被問進度   

Currently under review. We are waiting for API clarification and contract update before FE verification.    

2. 被問為什麼卡住

The blocker is not implementation. The dependency is API contract confirmation.            

3. 被問什麼時候完成    

After API contract update, FE will verify and then close the review.

# Notes
- 臨場表達只是需要練習，不代表你的判斷不夠快。你可以把這些常見場景做成幾個固定句型，之後會自然很多。
